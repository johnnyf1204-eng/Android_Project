# GymRank – Competitive Workout Tracker 

### Overview
**GymRank** is a fitness tracking app that gamifies your workouts by ranking users based on their performance.  
Each user can log exercises, sets, reps, and weights, while the app calculates a performance score using their body weight and total volume lifted.  
Multiple users share a common leaderboard — making training fun, competitive, and motivating.  
All data is stored locally for complete offline access.

---

###  Features
-  **Multi-user local profiles**
-  **Workout tracking** (exercises, sets, reps, weights)
-  **Performance scoring system** (based on body weight & total volume)
-  **Shared local leaderboard** for friendly competition
-  **Workout history** per user
-  **Workout streaks** (track consecutive training days)

---
 Technologies & Android Concepts
- **Language:** Kotlin  
- **Architecture:** MVVM + Repository Pattern  
- **UI Components:** RecyclerView, Fragments, Navigation Component  
- **Data Storage:** Room Database (local persistence)  
- **Navigation:** Jetpack Navigation  
- **LiveData & ViewModel:** for reactive UI updates  

---
Target Audience
Gym enthusiasts and fitness-focused individuals who want to track and compare their performance with friends —  
perfect for small groups sharing one device or competing offline.

---

###  Added Value
Unlike traditional gym trackers, **GymRank** adds a competitive twist.  
It motivates users through a **shared leaderboard** and a **performance formula** that fairly compares strength across different body weights.  
The app works fully **offline**, focusing on privacy, simplicity, and motivation.
